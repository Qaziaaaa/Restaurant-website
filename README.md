# Restaurant Website

Full-stack restaurant platform with menu management, table reservations, and online ordering system. Built with React, TypeScript, and Vite.

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Build Tool**: Vite
- **Features**: Menu display, reservation system, online ordering

## Getting Started

1. Clone the repository
2. Install dependencies: \
pm install\
3. Set the \GEMINI_API_KEY\ environment variable
4. Run development server: \
pm run dev\
