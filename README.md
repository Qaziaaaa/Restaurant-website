# Savoria - Advanced Restaurant Website

Savoria is a premium, high-end restaurant website built with React, Tailwind CSS, and Framer Motion. It features a modern, responsive design with advanced frontend functionalities.

## ✨ Key Features

- **Premium Aesthetics**: Glassmorphism, smooth animations, and a curated color palette (Savoria Orange & Ink).
- **AI-Powered Imagery**: Custom-generated AI portraits for our global team of master chefs.
- **Cart System**: Fully functional shopping cart using React Context API for state management.
- **Micro-interactions**: Interactive hover effects, floating elements, and smooth transitions.
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewports.
- **Modular Architecture**: Component-based structure for scalability and maintainability.

## 🚀 Technologies Used

- **React**: Frontend framework.
- **Tailwind CSS**: Utility-first CSS styling.
- **Lucide React**: Premium iconography.
- **Framer Motion**: Advanced animations and micro-interactions.
- **Context API**: Global state management for the ordering system.

## 🛠️ Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
